BGS - Backup Game Saves
-----------------------
Initial app design by Ekianjo up until v0.2.4.2
Code refinement and additions since v0.2.5.0 by kickass
-----------------------
Stop waiting for your gamesaves to get lost.
Back them up right now!

Once you stored your backup in a safe place
you can easily restore them with this tool.

v0.2.5.0 (Aug 2014):
-----------------------
added google drive support
introduced a backup logging file (bgs.log)
backups on a per game/emulator basis
restore functionality

Thank you for using BGS on your Open Pandora!
